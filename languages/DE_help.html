Bot-Trap Tool

Dieses Tool ermöglicht die einfache Kontrolle der LOG Dateien, die von der Datei page.restrictor.php des Bot Trap Projekt angelegt werden.

Das Bot-Trap Tool erwartet, dass sich die page.restrictor.php im Wurzelverzeichnis der Domain befindet. Der Aufruf sollte über die config.php von WebsiteBaker bzw. LEPTON CMS erfolgen:

// init Page Restrictor first...
if (file_exists(dirname(__FILE__).'/page.restrictor.php')) {
  $logfile = dirname(__FILE__).'/modules/dbbottrap/logs/'.date('Ymd').'.log';
  define('PRES_LOG_FILE', $logfile);
  require_once(dirname(__FILE__).'/page.restrictor.php'); 
}

Darüber hinaus muss das Verzeichnis /logs existieren, in diesem werden die LOG Dateien abgelegt.
